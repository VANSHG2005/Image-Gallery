# Image Gallery

A simple, responsive image gallery web application built using HTML, CSS, and JavaScript. This project showcases a collection of images in a visually appealing layout, complete with responsive design and basic interactivity.

## Features

- 📱 **Responsive Design** – Optimized for all screen sizes including desktop, tablet, and mobile devices.
- 🖼️ **Image Grid Layout** – Clean and organized display of images using CSS grid or flexbox.
- ⚡ **Lightweight and Fast** – Minimal dependencies for quick load times.
- 🧭 **User-Friendly Interface** – Simple navigation and intuitive user experience.

## Technologies Used

- **HTML5** – Markup for structuring the gallery.
- **CSS3** – Styling for layout, responsiveness, and visual effects.
- **JavaScript** – Adds interactivity and dynamic behaviors (e.g., image preview modal or hover effects).

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- A code editor (optional, e.g., VS Code)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/VANSHG2005/Image-Gallery.git
2. Navigate to the project directory:
   cd Image-Gallery

3. Open the index.html file in your browser:
   open index.html

   Or simply double-click index.html in your file explorer.
